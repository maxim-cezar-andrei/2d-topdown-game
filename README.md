# Shadow Heist - 2D Top-Down RPG (Java)

Joc 2D de tip RPG cu perspectiva top-down, plasat intr-un oras 
cyberpunk distopic. Jucatorul controleaza un ninja care se 
infiltreaza in cladirile unei megacorporatii, colecteaza dovezi 
si evita paznicii automatizati.

## Tehnologii folosite
- Java (Swing)
- MySQL (salvare/incarcare progres)
- Tiled (harti CSV)

## Functionalitati
- Miscare personaj cu animatii (Idle, Run, Attack)
- Sistem AI pentru urmarirea si atacul inamicilor
- Camera dinamica care urmareste personajul
- Coliziuni bazate pe tilemap
- Sistem de combat cu hitboxuri
- Salvare si incarcare progres in baza de date MySQL
- 3 niveluri diferite sub forma de labirint
- HUD cu HealthBar si Scor
- Meniu principal si meniu de pauza

## Structura proiectului
- `src/` - codul sursa Java
- `assets/` - sprite-uri si tilemap-uri
